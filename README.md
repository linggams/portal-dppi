# DPPI

Aplikasi pengajuan barang (Purchasing) dan tiket gangguan IT — Next.js + PostgreSQL + Prisma.

## Dokumentasi

- [SETUP.md](./SETUP.md) — instalasi development & database
- [SERVICE-PM2.md](./SERVICE-PM2.md) — production Windows (PM2)
- [docs/architecture.md](./docs/architecture.md) — struktur modul & level user

## Perintah cepat

```bash
pnpm install
pnpm dev          # http://localhost:2000
pnpm build
pnpm pm2:start    # production (lihat SERVICE-PM2.md)
```
